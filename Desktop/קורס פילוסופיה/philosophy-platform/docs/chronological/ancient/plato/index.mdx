# אפלטון (428-348 לפנה"ס)

## הקשר היסטורי

אפלטון נולד באתונה משפחה אריסטוקרטית. הכיר את סוקרטס כשהיה צעיר וחד רושם בו. לאחר הוצאתו להורג של סוקרטס, אפלטון התחייב לשנות את העולם דרך **פילוסופיה**.

ייסד את **האקדמיה** — מוסד למחקר שקיים למעלה מ-900 שנה! כל בן־משפחה מעל פתח של האקדמיה כתוב: **"מי שלא יודע גיאומטריה — אל יכנס"**.

## המערכת הפילוסופית

אפלטון בנה **עולם דואלי** — עולם הגופנים וחומרים (שגוי וחלוף) מול **עולם הצורות** (אלחי וייצחי).

## תרומות עיקריות

- **תיאוריית הצורות:** פילוסופיה לא שונה — מטמטית, מופשטת, נצחית
- **דיאלוג סוקרטי:** שיטת ההנחיה (maieutic) — מדריך אחר כדי למצוא אמת
- **הפילוסוף-מלך:** ראיית בלטת תוך־תיקת של משלה וידע בפוליטיקה
- **השקתנו של הנשמה:** טען בהתקיימות של הנשמה קודם להגוף

## הדיאלוגים של אפלטון

אפלטון לא כתב מחברות מדעיות. הוא כתב **דיאלוגים** — שיחות בין סוקרטס לחוקרים אחרים. זו שיטה דרמטית ושונה המעודדת את הקורא למצוא אמת בעצמו.

> "אני יודע שאני לא יודע — וזה הופכני לחכם מאנשים שחושבים שהם יודעים." — סוקרטס, בדברי אפלטון

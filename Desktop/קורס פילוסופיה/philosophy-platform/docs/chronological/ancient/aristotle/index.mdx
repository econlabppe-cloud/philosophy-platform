# אריסטו (384-322 לפנה"ס)

## הקשר היסטורי

אריסטו נולד בסטגירה שבמקדוניה, היה תלמידו של אפלטון ומורו של אלכסנדר הגדול. ייסד את הליקיאון באתונה — מוסד מחקר שהיה הבסיס לאוניברסיטאות המודרניות.

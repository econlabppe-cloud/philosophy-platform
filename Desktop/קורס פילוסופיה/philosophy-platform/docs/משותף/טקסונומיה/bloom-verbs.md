# טקסונומיית בלום

## רמות הטקסונומיה

### 1. ידע (Remember)
- הגדרה, זיהוי, רשימה, שינון

### 2. הבנה (Understand)
- הסבר, תיאור, סיכום, פרשנות

### 3. יישום (Apply)  
- שימוש, יישום, הדגמה, פתרון

### 4. ניתוח (Analyze)
- השוואה, ארגון, פירוק, זיהוי מרכיבים

### 5. הערכה (Evaluate)
- שיפוט, ביקורת, הצדקה, טיעון

### 6. יצירה (Create)
- עיצוב, בנייה, פיתוח, יצירה חדשה

---

## המשפך הפדגוגי

### שלב 1: הקרקע → הבנה
הלומד מבין את הקונפליקט הפילוסופי ואת ההקשר ההיסטורי.

### שלב 2: הכלי → יישום + ניתוח  
הלומד מיישם את הידע כדי לנתח טענה מודרנית.

### שלב 3: העומק → הערכה + יצירה
הלומד מעריך את עמדותיו ויוצר תובנות אישיות.
